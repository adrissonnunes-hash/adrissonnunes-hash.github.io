Como identificar retenção de ISS no XML da NFS-e

8 min • Conteúdo prático
Espaço reservado para anúncios
Quando sua conta for aprovada (ex.: AdSense), substitua este bloco pelo código do anúncio.

Nem todo XML de NFS-e tem o mesmo layout (padrões variam), mas a lógica é sempre a mesma: existe um indicador/campo que diz se o ISS foi retido e, quando houver retenção, os valores relacionados precisam fazer sentido.

O que conferir no XML

  • Indicador de retenção: campo que indica se o ISS está retido.
  • Valor de ISS/retido: quando existir, confira se o cálculo bate com a base.
  • Valores federais: IRRF/CSLL/PIS/COFINS podem aparecer como retenções separadas.

Conferência segura

Se você confere em volume, o ideal é marcar visualmente “tem retenção” e “não tem retenção”, e só depois totalizar. Isso reduz erro de soma.
